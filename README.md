#  Zepto SQL Data Analysis Project

##  Overview

This project analyzes product data from Zepto using SQL.
The goal is to extract business insights such as pricing trends, discounts, stock availability, and revenue estimation.

---

##  Tools Used

* MySQL
* SQL (Joins, Aggregations, CASE, GROUP BY)

---

## Key Analysis Performed

### 1. Top Discounted Products

Identified top 10 products with highest discount percentage.

### 2. High MRP but Out of Stock

Analyzed expensive products that are currently unavailable.

### 3. Estimated Revenue by Category

Calculated total revenue based on selling price and quantity.

### 4. Pricing Filter

Filtered products where price > ₹500 and discount < 10%.

### 5. Top Categories by Discount

Found categories offering highest average discounts.

### 6. Price per Gram Analysis

Calculated value per gram to identify best-value products.

### 7. Product Weight Segmentation

Classified products into Low, Medium, and Bulk categories.

### 8. Inventory Weight Analysis

Computed total inventory weight per category.
